# Calculator-Jetpack-Compose
Beginner calculator app using Kotlin and Jetpack Compose
This is a simple Calculator Android application built using Kotlin and Jetpack Compose.
The app performs basic arithmetic operations like:

Addition ➕

Subtraction ➖

Multiplication ✖️

Division ➗

This project is made for learning Jetpack Compose basics, state handling, and UI building in Android.
